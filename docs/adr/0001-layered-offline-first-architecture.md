# ADR 0001：离线优先分层架构

- 状态：接受
- 日期：2026-07-11

## 决策

规则核心采用纯 TypeScript；React/Vite 作为 Web UI；平台能力通过 application ports 注入。未来 Electron 是独立壳和适配器，不允许 renderer 直接依赖 Node/Electron。

## 理由

游戏的长期价值集中在可测试、可重放的规则系统。将规则与界面、浏览器和桌面壳隔离，可共享测试、降低跨平台成本，并防止预览与实际执行分叉。

## 后果

新增平台能力需要端口与契约测试，初期文件稍多；换来确定性、可移植性和可审计依赖方向。
