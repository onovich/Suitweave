# Suitweave Phase 1 领域契约

本文件冻结 Phase 1 的最小公共类型边界。产品名为“织花牌 / Suitweave”；《三盘连彩》仅作为历史 GDD 来源。

## 唯一事实来源

`GameState` 持有 seed、版本化 `Ruleset` 与三张 `Board`。每张棋盘的 `Cell` 分离数字层（`PlayingCard`）、墨色层（`InkColor`）、标记层与锁定状态。后续规则只从此状态读取，不从 UI 或平台 API 读取。

## 命令与事件

`Command` 只描述一次领域意图；领域转换将返回新状态与 `DomainEvent[]`。当前轮只冻结 `place-number`、`place-ink` 和拒绝事件的结构，转换逻辑将在后续轮次实现。
