# Phase 5 automation baseline

Status: AUTOMATED_READY. Simulation policies are non-human tools and must not be used to infer player understanding, session length, completion rate, or balance quality.

Each run is reproducible from `balance-v1`, its commit, `BalanceRunManifest`, frozen seed catalog, policy, and step cap. The aggregator records sample size, means, p10/p50/p90, completion/crown rates, and abnormal seeds rather than reporting only an average.

No balance parameter changes are authorized until anonymous real-player evidence passes the Phase 5 import and privacy checks.
